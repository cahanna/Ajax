Ajax Applications




Application Program Interfaces (API) are a set of rules which make a conversation possible between two different programs. A developer creates an API on a server to allow a client to access it. Representational State Transfer are the rules that say you should be able to get a resource when you link to a specific URL. Each URL is called a request and the data sent back is called a response.

One of my biggest interests is comic books. Marvel superheroes are more popular now than ever in film and tv. If I were creating a comic book app I would use a REST API called Superhero API. It has tons of Super Heroes and Villians data from all comic book universes. It has a request and response format JSON. It requires an API Key which can only be obtained by linking a Facebook account. Here is a link with more information about it. Superhero API. This could be a great resource for someone who wants to create a database about comic book characters.
